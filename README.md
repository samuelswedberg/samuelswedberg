## Featured Project: Sim Racing Wheel System

Year-long senior capstone at NDSU — a full sim racing force feedback system (wheelbase, steering wheel, pedals) built from scratch. Earned top faculty and industry feedback at the NDSU Senior Design Expo.

- **Firmware** — STM32 with FreeRTOS for real-time motor control, USB OTG HID device driver, and peripheral management
- **Communication** — Multi-device CAN bus network across 3 custom two-layer PCBs
- **Software** — Python PC application for user settings and game telemetry over serial
- **Hardware** — Mechanical components modeled in Fusion 360, 3D-printed enclosures and structural parts (PA6-CF & ABS), full PCB design and system integration
- **Validation** — Oscilloscope, logic analyzer, and multimeter for hardware debugging and signal verification
